# NewsSense 🧠📰

NLP-powered Flask web application for news analysis.

## Features
- News input via text, file, or URL
- Summarization using Sumy (LSA)
- Sentiment analysis using TextBlob
- Named Entity Recognition using spaCy
- News classification using TF-IDF + Naive Bayes

## Tech Stack
Python, Flask, NLTK, spaCy, scikit-learn, BeautifulSoup

## How to Run
```bash
pip install -r requirements.txt
python app.py
